Part 1 — Zendesk API/SDK to Replace with New CRM
These are direct Zendesk product dependencies. The new CRM tool must provide equivalent capabilities.

Zendesk Products In Use

  • Support (Tickets API v2) — Ticket CRUD, comments, attachments, tags, custom fields, incremental exports
  • Users API v2 — User creation, search, tag management, external ID mapping
  • Help Center API — Article fetching, section/category traversal, FAQ content for chatbot embeddings
  • Messaging SDK — In-app chat on iOS (ZendeskSDKMessaging) and Android (v2.31.0)
  • Support SDK — Native ticket views on iOS and Android (v5.4.0)
  • AnswerBot SDK — AI-powered article suggestions in mobile apps (v3.3.0)
  • Webhooks (Outbound) — Ticket lifecycle events pushed to incoming-webhooks service
  • Apps Framework (ZAF) v2 — Custom sidebar widget in agent dashboard

New CRM Must Support

  • Ticket/Case Management API — Create, read, update tickets; add comments; manage attachments
  • Contact/User API — Create users, search, tag/label management, external ID sync
  • Knowledge Base API — Fetch articles programmatically for chatbot RAG pipeline
  • Mobile Chat/Messaging SDK — iOS + Android native SDK with JWT auth and push notifications
  • Native Ticket UI SDK — In-app ticket list, detail, and creation views
  • AI/Bot SDK — Article suggestion engine in mobile apps
  • Outbound Webhooks — Event triggers for ticket creation, status changes, replies
  • Agent Sidebar / App Framework — Custom iframe widget showing customer data in agent view
  • Incremental Export API — Cursor-based bulk ticket retrieval for sync
  • CSAT Survey Support — Post-resolution satisfaction ratings
